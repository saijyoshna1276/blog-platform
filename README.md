# Blog Platform with Comments

## Project Description

A full-stack blogging platform where users can register, log in, create blog posts, edit posts, delete posts, and interact through comments.

## Features

* User Registration
* User Login Authentication
* Create Blog Posts
* Edit Blog Posts
* Delete Blog Posts
* Comment on Posts
* RESTful APIs
* Database Integration

## Technology Stack

### Frontend

* React
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT
* bcrypt
